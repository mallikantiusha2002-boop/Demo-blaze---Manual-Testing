## Project Overview
Complete end-to-end manual testing of the DemoBlaze e-commerce demo application  
Covered 8 core modules with real execution:

- User Registration  
- Login & Logout  
- Product Browsing & Categories  
- Shopping Cart  
- Checkout & Place Order  
- Contact Form  
- Navigation Menu  
- (Search Functionality – fully tested & reported as missing)

88 Test Cases Designed & Executed  
26 Failed → 19 Unique Real Defects Found  
Overall Pass Rate: 70.45%
